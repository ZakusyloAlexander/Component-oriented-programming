# Лабораторна робота 4 — Matching Emojis (роутинг, контекст, стилізація)

Ця версія гри базується на Лаб 3, але додає:

- динамічний **роутинг з id користувача** (`/users/:userId/...`) за допомогою `react-router-dom`;
- **стейт-менеджмент через контекст** для налаштувань та останньої статистики;
- стилізацію за допомогою **styled-components** (обгортка застосунку) + адаптивні CSS-стилі.

## Основні маршрути

- `/` — головна, форма для введення `userId`.
- `/users/:userId/settings` — екран налаштувань та старту гри для конкретного користувача.
- `/users/:userId/game` — саме поле гри.
- `/users/:userId/results` — підсумки останнього туру для цього користувача.

## Технології

- **React 18 + Vite**
- **react-router-dom** — роутинг із параметром `:userId`
- **React Context** (`GameSettingsContext`) — налаштування гри + останні результати
- **react-hook-form + Yup** — форма налаштувань із валідацією
- **styled-components** — стилізація оболонки додатку (`Shell`, `Container`, `HomeCard`)
- Класичний CSS (`src/styles/base.css`) для грідів, карток і статистики.
