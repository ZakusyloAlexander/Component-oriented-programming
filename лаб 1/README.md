# Лабораторна робота 1 — Matching Emojis (каркас)

Ця папка містить каркас застосунку для гри **Matching Emojis**. Реалізовано лише базові сторінки та плейсхолдери без бізнес-логіки та роутингу, як вимагає завдання.

## Структура

- `index.html` — точка входу, підключає стилі та модуль JavaScript.
- `styles/base.css` — базові стилі, грід, теми для сторінок.
- `src/app.js` — ядро застосунку, перемикає між сторінками без роутера.
- `src/main.js` — ініціалізація застосунку.
- `src/state/gameState.js` — початкові стани та заготовки для статистики/складності.
- `src/components` — перевикористовувані компоненти для сторінок:
  - `StartScreen.js`
  - `GameBoard.js`
  - `ResultsScreen.js`
- `package.json` — налаштований для запуску проєкту через `npm` з використанням Vite.

## Як запустити

### Варіант 1 — напряму в браузері

1. Відкрийте файл `index.html` у браузері (подвійний клік або через розширення Live Server).
2. Перевірте навігацію між сторінками кнопками.

### Варіант 2 — через npm (Vite)

1. У терміналі перейдіть у папку `лаб 1`:
   - `cd "C:\Учеба\компонентно оріентовне програмування\лаб 1"`
2. Встановіть залежності:
   - `npm install`
3. Запустіть dev-сервер:
   - `npm run dev`
4. Відкрийте посилання `http://localhost:5173/` (або яке покаже Vite).
