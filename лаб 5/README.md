# Лабораторна робота 5 — Matching Emojis (Redux Toolkit)

На базі Лаб 4 реалізовано:

- єдиний **Redux Toolkit store** для налаштувань гри та таблиці результатів;
- маршрути `/`, `/users/:userId/settings`, `/users/:userId/game`, `/users/:userId/results`;
- збереження даних у `localStorage` через слайси;
- стилізація (styled-components + CSS) та портал з результатами.

## Технології

- React 18 + Vite
- React Router DOM 6
- Redux Toolkit + React-Redux
- react-hook-form + Yup
- styled-components

## Запуск

```powershell
cd "C:\Учеба\компонентно оріентовне програмування\лаб 5"
npm install
npm run dev
```

Після запуску відкрийте адресу Vite (орієнтовно `http://localhost:5175/`).
