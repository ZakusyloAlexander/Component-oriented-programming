# Лабораторна робота 3 — Matching Emojis (налаштування + портали)

Ця версія продовжує попередні лабораторні, але тепер застосунок переписано на React із підтримкою:

- форми налаштувань гри на **react-hook-form** + **Yup** із синхронізацією в `localStorage`;
- кастомного контексту для поширення налаштувань між сторінками;
- повної бізнес-логіки гри на хуках;
- портального діалогового вікна з підсумками та кнопками «Наступний тур» / «Повторити».

## Структура

- `index.html` — містить `#root` для застосунку та `#modal-root` для порталів.
- `src/main.jsx` — початок React-застосунку.
- `src/App.jsx` — навігація між екранами (старт, гра, результати).
- `src/contexts/GameSettingsContext.jsx` — налаштування гри з локальним станом + localStorage.
- `src/hooks/useEmojiSession.js` — ігрова логіка (колода, таймер, матчинг).
- `src/components/SettingsForm.jsx` — форма з `react-hook-form` / `yup`.
- `src/components/GameBoard.jsx` — поле гри, підключає `useEmojiSession`.
- `src/components/ResultsModal.jsx` — портал із кнопками дій.
- `src/components/StatsPanel.jsx` — віджет для статистики.
- `src/state/constants.js` — набір емодзі та пресети.
- `src/styles/base.css` — базові стилі (підключаються в `App.jsx`).

## Що реалізовано

1. **Налаштування гри** — форма дозволяє обрати рівень складності, кількість пар та затримку приховування карток. Дані зберігаються у `localStorage`, тож після перезавантаження останні опції підтягуються автоматично.
2. **Ігрова логіка на хуках** — `useEmojiSession` керує створенням колоди, рахунком ходів, таймером та обчисленням точності.
3. **Портальне діалогове вікно** — після завершення з’являється модальне вікно з підсумком, можливістю зіграти той самий раунд або повернутися до налаштувань для нового турніру.
